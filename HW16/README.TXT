Задание разбито на 2 модуля: HW16 и HW16.1
HW16 - клиентская часть и вся логика обмена сообщениями.
HW16.1 - маленькая серверная часть и запуск клиентов.

Запуск клиентов выполняется аналогично примеру из HW16.1 ServerMain.

Для работы серверной части создана карта карт - сначала поиск по типу клиента, потом по id. Не очень красиво, но позволяет делать некоторую балансировку.
Балансировка для БД создана как рандомный выбор из набора.

В клиентской части попытался сохранить возможность работать нескольким службам на одной локальной службе сообщений (по аналогии с HW15).
